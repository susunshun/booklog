# booklog
- 読書管理アプリ
- バーコード読み取りで簡単登録
- 画像認識でも登録可能（未実装）
- UIの勉強で作ったのでデータの永続化はしてないです。。。

## 使用サービス
- 楽天books API（書籍情報取得）
- monacaプラグイン（バーコード読み取り）
- docomo API（画像認識）
